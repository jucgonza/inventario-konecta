DESCRIPCION DEL PROYECTO:
Este proyecto fue desarrollado especificamente para cumplir con el requisito de prueba practica, para optar por el cargo de desarrollador PHP en la empresa Grupo Konecta.

Se basa en el modelo MVC y esta construido sobre el framework PHP Codeigniter v3.1.1 en el backend, y usando el framework CSS Bootstrap v5 para el frontend.

INSTALACIÓN & CONFIGURACIÓN:
No es necesario instalar plugins o dependencias de terceros, sin embargo se debe garantizar lo siguiente:
PHP >= 5.6
MYSQL MariaDB >= 10.1.44

PARA EMPEZAR:
- El archivo konecta_dump.sql en la raiz contiene el dump de la base de datos usada para este proyecto.
- Dentro de la carpeta application/config encontrará el archivo database.php en el cual debe llenar los items 'username', 'password' y 'database' correspondientes a su configuración de base de datos local.
- En caso de usar un nombre de fichero distinto al original (inventario) debera dirigirse a la carpeta application/config en donde encontrará el archivo config.php en el cual deberá reemplazar la instrucción $config['base_url'] con la correspondiente a su ambiente local.
- Inicie su servidor, abra el proyecto en un navegador y listo.
